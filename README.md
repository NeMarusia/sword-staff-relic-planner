# Sword x Staff: база реликвий

Минимальный статический прототип подбора реликвий.

## Что умеет

- Загружает публичный JSON реликвий Loot & Waifus.
- Показывает картинки реликвий по `https://lootandwaifus.com/treasures/swordxstaff/<icon>`.
- Даёт отметить реликвии, которые есть у игрока.
- Сохраняет отмеченные реликвии и уровни в `localStorage`.
- Группирует реликвии по месту получения из открытой базы.
- Считает базовые рекомендации под Чародея, Дуэлянта, Рыцаря и Мудреца только из отмеченных реликвий.
- Переводит интерфейс, элементы, статы, редкость и часть названий реликвий через локальный словарь.

## Как открыть

Можно открыть `index.html` напрямую в браузере.

Или запустить локальный сервер:

```bash
python3 -m http.server 4177
```

## Важно

Сейчас это эвристический ранжировщик, а не точный калькулятор боя. Формула учитывает:

- вес основных статов под класс;
- рекомендуемые элементы класса;
- редкость;
- сетовые бонусы;
- введённый уровень прокачки.

Дальше нужно уточнять веса по гайдам и реальным тестам.

Русские названия реликвий не приходят из публичного JSON, поэтому словарь `RU.name` в `app.js`
нужно пополнять по скриншотам или другой русскоязычной выгрузке клиента.
